The open-source python package of "Graph-based Leaf¨CWood Separation Method for Individual Trees Using Terrestrial Lidar Point Clouds".
The main entrance is "GBSeparation.py"; Other 3rd party packages (numpy, networkx,sklearn,open3d ...) are needed. 
The code is compatible with Python 3.7.
The input data should be a single tree 3D point cloud and no large gaps in existence.
